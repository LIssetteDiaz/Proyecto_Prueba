Acerca de mi

Apasionada por el desarrollo web y el diseño de experiencias digitales únicas.

Área de especialización

Me desempeño principalmente en el desarrollo web frontend, enfocándome en la construcción de interfaces intuitivas y responsivas. Me interesa que cada detalle visual y cada interacción estén pensados para mejorar la experiencia del usuario y transmitir una identidad clara en cada proyecto.

Tecnologías y lenguajes

Trabajo con herramientas y tecnologías como HTML5, CSS3, JavaScript y Bootstrap, además de conocimientos en Font Awesome, Flexbox y Grid Layout para lograr estructuras adaptables y modernas. También utilizo Live Server y otros entornos de desarrollo que facilitan la visualización y prueba de proyectos en tiempo real.

Logros y enfoque

A lo largo de mis proyectos, he logrado diseñar y desarrollar páginas web funcionales, centradas en la usabilidad y la estética, manteniendo siempre una atención especial al detalle. Mi meta es seguir evolucionando como desarrolladora, integrando nuevas tecnologías y tendencias para ofrecer resultados cada vez más sólidos y creativos.

Mis trabajos

Autenticación de usuarios

Proyecto desarrollado en Ruby on Rails que implementa un sistema completo de autenticación de usuarios utilizando la gema Devise. Incluye registro, inicio de sesión, recuperación de contraseñas y control de acceso, ofreciendo una base sólida para aplicaciones web seguras y escalables.

Proyecto para gestión de un almacen

Aplicación web desarrollada en Django (Python) para la gestión y administración de un almacén. Permite controlar productos, proveedores, pedidos y recepción de mercadería mediante un sistema CRUD completo. El proyecto busca optimizar la organización interna y automatizar los procesos de registro y control de inventario.

Proyecto de título

Desarrollado en Django (Python) e integrado con JavaScript, este proyecto fue creado como parte del portafolio de título para la presentación formal del trabajo final. Incluye componentes funcionales y visuales que reflejan las habilidades adquiridas en desarrollo web y la aplicación práctica de herramientas modernas.
